บทคัดย่อ(ไทย) ผลจากการเติบโตอย่างรวดเร็วในทางอุตสาหกรรม ทำให้ความซับซ้อนในกระบวนการ ผลิตเพิ่มมากขึ้น ดังนั้นระบบควบคุมแบบเดิมที่ใช้จะไม่สามารถให้ผลการควบคุมที่ดีได้ เนื่องจากข้อจำกัดในเรื่องจำนวนกระบวนการที่สามารถควบคุมได้และเรื่องการควบคุม กระบวนการในเวลาจริง ระบบควบคุมแบบใหม่ที่นิยมใช้คือระบบสกาดา แต่การพัฒนาที่ เกิดขึ้นในระบบนี้มักเน้นทางด้านการติดต่อสื่อสารข้อมูล ในขณะที่ตัวควบคุมที่ใช้ในระบบ มักเป็นแบบธรรมดา เช่น แบบเปิด/ปิด ดังนั้นงานวิจัยนี้จึงมีวัตถุประสงค์ที่จะสร้างต้นแบบ ควบคุมสกาดาที่มีชนิดตัวควบคุมหลายชนิดให้เลือกใช้ การสื่อสารข้อมูลภายในระบบที่ พัฒนาขึ้นจะเป็นไปตามมาตรฐาน RS-485 และข้อมูลภายในระบบจะถูกส่งผ่านในรูปแบบ ฮาล์ฟ-ดูเพล็กซ์ ซึ่งใช้สายส่งสัญญาณ 2 เส้น และสามารถส่งผ่านข้อมูลได้ในระยะทางสูงสุด 4000 ฟุต และสามารถเชื่อมต่ออุปกรณ์เข้ากับระบบได้สูงสุด 32 อุปกรณ์ ระบบควบคุมนี้จะ ประกอบด้วยหน่วยควบคุมระยะไกล 2 หน่วยและหน่วยประมวลผลกลาง 1 หน่วย ซึ่งสามารถ ใช้งานได้กับตัวควบคุม 3 แบบ คือ GMC ฟัซซี่และพีไอ กระบวนการที่นำมาใช้ทดสอบการ ทำงานของต้นแบบควบคุม คือ กระบวนการระเหย ซึ่งมีตัวแปรควบคุม 3 ตัว คือ ระดับของ เหลวภายในเครื่องแยก (L2) ความเข้มข้นของผลิตภัณฑ์ (X2) และความดันภายในเครื่อง ระเหย (P2) ตัวควบคุมฟัซซี่จะถูกออกแบบตามขั้นตอน NFC ซึ่งจะได้เป็นตัวควบคุมแบบ ฟัซซี่รูลเบส ทั้งนี้วิธีการ Fuzzy c-Means Clustering และ Partition Coefficient ได้ถูกนำ มาประยุกต์ในขั้นตอนของการแบ่งกลุ่มข้อมูล อีกทั้งได้มีการพัฒนาตัวควบคุมฟัซซี่ให้มีการ ใช้รูปแบบของ Membership Function ที่แตกต่างกัน 2 แบบ คือ แบบสามเหลี่ยมและแบบ ทรงระฆัง เพื่อศึกษาผลกระทบของชนิด Membership Function ที่มีต่อประสิทธิภาพการ ควบคุมและการพัฒนาตัวควบคุมฟัซซี่บนต้นแบบควบคุมสกาดาที่พัฒนาขึ้น โดยตัวควบคุม ฟัซซี่แบบที่ 1 จะใช้กฏการควบคุม 10 ข้อ และ 30 ข้อ สำหรับการรบกวนระบบแบบเปลี่ยน แปลงค่าตัวแปรรบกวนและแบบการเปลี่ยนค่าเป้าหมายของระบบ สำหรับตัวควบคุมฟัซซี่ แบบที่ 2 จะใช้กฏการควบคุม 22 ข้อสำหรับการรบกวนระบบแบบเปลี่ยนแปลงค่าตัวแปร รบกวน และ 23 ข้อสำหรับการรบกวนระบบแบบเปลี่ยนแปลงค่าเป้าหมาย ผลการควบคุม กระบวนการของตัวควบคุมฟัซซี่ที่ใช้แบบ Membership Function ที่ต่างกันแสดงให้เห็นว่า Membership Function แบบสามเหลี่ยมให้ผลการควบคุมที่ราบเรียบกว่า เนื่องจากค่าความ เป็นสมาชิกที่ได้จากการใช้ Membership Function แบบสามเหลี่ยมจะปรากฏจะไม่ทำให้เกิด การใช้กฏการควบคุมที่ขัดแย้งกันในเวลาเดียวกัน สำหรับผลการควบคุมจากตัวควบคุม GMC ฟัซซี่และพีไอแสดงว่า ตัวควบคุม GMC ให้ผลการควบคุมที่ดีที่สุด เนื่องจากลักษณะสมการ ของ GMC ได้มีการรวมลักษณะของ Feedforward และการลดผลจาก Coupling เอาไว้ใน แบบ Feedback แล้ว สำหรับตัวควบคุมแบบฟัซซี่ มีเพียงความสามารถในการลดผลจากการ Coupling เท่านั้น จึงให้ผลการควบคุมที่ดีรองจากแบบ GMC อีกทั้งตัวควบคุมฟัซซี่ยังใช้ ข้อมูลที่ได้จากการทำ Simulation ของเครื่องระเหยที่ใช้ตัวควบคุม GMC รวมทั้งขั้นตอนของ ฟัซซี่ที่ใช้ในการควบคุมจะทำให้ประสิทธิภาพในการควบคุมของตัวควบคุมฟัซซี่ลดต่ำลง ใน ขณะที่ตัวควบคุมพีไอให้ผลการควบคุมที่ต่ำที่สุดในการเปรียบเทียบ เพราะเป็นการควบคุมแบบ Feedback ธรรมดา ซึ่งไม่สามารถลดผลกระทบที่เกิดจาก Coupling และไม่มีการรวมลักษณะ ของ Feedforward ไว้ในการควบคุม อย่างไรก็ตามลักษณะของ Feedforward สามารถถูก กำหนดเข้าไปในตัวควบคุมฟัซซี่ได้ด้วยการเปลี่ยนลักษณะของข้อมูลที่ใช้ในการพัฒนาตัว ควบคุมฟัซซี่
บทคัดย่อ(English) As the result of rapidly growth in the industries, the complexity of production process is also greater than before. The conventional control system can no longer provide acceptable results due to its capability limitation in controlling the multi-process and the real time control. SCADA system is introduced to these problems. However, the development in SCADA only tends to develop the part of data acquisition while controller is usually the conventional one such as ON/OFF. This research, therefore, focuses on the set up of a SCADA prototype, which has more alternative controllers. Information communication of the developed system is based on the RS-485 standard and the data are transmitted in the half-duplex communication with 2-wire transmission line in the maximum 4000 feet long. Maximum 32 units can be, connected to this SCADA system. The prototype developed in this research consists of 2 RTUs and one CMS with 3 types of controller; GMC, fuzzy, and Pl. The test case; evaporator, has 3 controlled variables; separation level (L2), product concentration (X2), and operating pressure (P2). Fuzzy controller was developed by the NFC steps; the fuzzy rule based controller. Fuzzy c-means clustering and partition coefficient were applied in the data partitioning step. In addition, two types of membership function; triangular and bell shape, were used to study the effect of the membership function on the control performance, and the application of fuzzy rule based controller on SCADA system. There are 10 rules for load change and 30 rules for set point change in the 1(st) fuzzifier. For the 2(nd) fuzzifier, it needs 22 and 23 rules for the load and set point change. Triangular gives a smoother control because it has no internal conflicting rules as in the bell shape membership function. The comparison of these controllers indicates that GMC gives the best control because it by nature includes the feedforward, and decoupling characteristics although it uses only usual feedback control loop. For fuzzy controller, only the decoupling is included so the control result cannot be better than GMC. However, this fuzzy controller was trained by GMC since it uses the numerical data from the simulation of evaporator with GMC. The control efficiency can be lost due to the mapping steps in fuzzy control. PI gives the worst result because it handles 3 common feedback loops simultaneously, so there still is an inherent coupling effect. Moreover, PI does not naturally provide the feedforward characteristic.
