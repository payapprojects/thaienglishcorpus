ใช้ Pages, Numbers และ Keynote กับ Box
ทำงานร่วมกับผู้อื่นแบบเรียลไทม์บนเอกสาร Pages, Numbers และ Keynote ที่จัดเก็บอยู่ใน Box 
นอกจากนี้ คุณยังสามารถดูตัวอย่าง แก้ไข และสร้างเอกสารใหม่แบบออนไลน์ที่ box.com ได้ด้วย
ทำงานร่วมกันบนเอกสารโดยใช้ Box กับอุปกรณ์ Apple และซอฟต์แวร์เวอร์ชั่นต่อไปนี้
iPhone, iPad หรือ Mac ที่มี iOS 12, iPadOS หรือ macOS Mojave หรือใหม่กว่า และ Pages, Numbers หรือ Keynote เวอร์ชั่น 10.1 
Box 4.3.0 หรือใหม่กว่าสำหรับ iPhone หรือ iPad, หรือ Box Drive สำหรับ Mac 1.10 หรือใหม่กว่า หรือ Box Edit สำหรับ Mac 4.0 หรือใหม่กว่า
หากต้องการทำงานร่วมกันบนเอกสาร Pages, Numbers และ Keynote แบบออนไลน์ที่ box.com คุณจะต้องใช้ Safari 9.1.3 ขึ้นไป หรือ Google Chrome สำหรับ Mac หรือ Google Chrome หรือ Microsoft Edge สำหรับพีซี
เริ่มต้นใช้งาน
ไปที่ box.com เพื่อสมัครบัญชี1 Box ส่วนบุคคล 
มีบัญชีอยู่แล้วใช่ไหม ถ้ามีอยู่แล้ว ให้ไปยังขั้นตอนถัดไป
หากคุณมีบัญชีส่วนบุคคล ให้ลงชื่อเข้าใช้ แล้วคลิกหรือแตะตัวย่อของคุณที่มุมขวาบนของเบราเซอร์ 
เลือก Account Settings (การตั้งค่าบัญชี) แล้วเลือก Integrations (การรวม) 
สำหรับบัญชีขององค์กร ผู้ดูแลระบบ Box Enterprise สามารถเข้าใช้การตั้งค่า Integration (การรวม) ได้ใน Admin Console (คอนโซลผู้ดูแลระบบ) > Settings (การตั้งค่า) > Apps (แอพ)
หากต้องการเปิดการทำงานร่วมกันบน iWork ให้เลือก iWork Collaboration (การทำงานร่วมกันบน iWork)
หากต้องการดูตัวอย่างเอกสาร iWork ใน Box ทางออนไลน์ ให้เลือก iWork Previews (การแสดงตัวอย่าง iWork)
ทำงานร่วมกันในเอกสาร
เมื่อคุณตั้งค่า Box และเปิดใช้งานการทำงานร่วมกันบน iWork คุณสามารถแก้ไขบนเอกสาร iWork ที่จัดเก็บไว้อยู่แล้วใน Box ร่วมกับผู้อื่นในแบบเรียลไทม์ได้โดยอัตโนมัติ
หากต้องการเริ่มต้นการทำงานร่วมกันบนเอกสารใหม่ ให้ย้ายหรือบันทึกเอกสารของคุณใน Box โดยทำดังนี้
บน Mac ให้ลากเอกสารของคุณไปยังโฟลเดอร์ Box ในแถบด้านข้างของ Finder 
หากไม่มี Box อยู่ในแถบด้านข้างของ Finder ให้เปิดแอพ Box จากโฟลเดอร์แอพพลิเคชั่น
บน iPhone หรือ iPad ให้ย้ายหรือคัดลอกเอกสารของคุณไปยัง Box โดยใช้ตัวจัดการเอกสารใน Pages, Numbers หรือ Keynote หรือใช้แอพไฟล์2
ออนไลน์ในบัญชี Box ของคุณ ลงชื่อเข้าใช้บัญชีของคุณ แล้วลากเอกสารไปยังหน้าจอ All Files (ไฟล์ทั้งหมด) 
หรือคลิก Upload (อัพโหลด) เลือก File (ไฟล์) แล้วเลือกเอกสาร
หากต้องการเชิญผู้อื่น ให้เปิดเอกสารของคุณจากโฟลเดอร์ Box คลิกหรือแตะปุ่มทำงานร่วมกัน จากนั้นเชิญบุคคลนั้นผ่านทาง box.com 
หลังจากที่คุณเริ่มทำงานร่วมกันแล้ว คุณสามารถจัดการเอกสารได้โดยทำดังนี้
หากต้องการดูว่าใครกำลังแก้ไขเอกสารอยู่ ให้คลิกหรือแตะปุ่มทำงานร่วมกัน 
หากต้องการจำกัดผู้ที่สามารถดูและแก้ไขเอกสารของคุณได้ ให้คลิกหรือแตะปุ่มทำงานร่วมกัน  แล้วแตะการตั้งค่าการแชร์
แก้ไขขณะออฟไลน์
คุณยังคงแก้ไขเอกสารที่แชร์ได้แม้จะไม่ได้เชื่อมต่ออินเทอร์เน็ต 
การแก้ไขที่คุณทำแบบออฟไลน์จะได้รับการบันทึกไว้เป็นเวลาอย่างน้อย 30 วัน และจะอัพโหลดไปยัง Box โดยอัตโนมัติเมื่อคุณกลับมาออนไลน์อีกครั้ง 
ผู้เข้าร่วมคนอื่นจะเห็นสิ่งที่คุณแก้ไขเมื่ออัพโหลดเรียบร้อยแล้ว 
คุณสามารถตรวจสอบสถานะการซิงค์ของเอกสารได้จากตัวจัดการเอกสารบน iPhone หรือ iPad หรือเลือกมุมมอง > แสดงสถานะการซิงค์บน Mac
หากต้องการส่งสำเนาเอกสารที่คุณแก้ไขให้ผู้อื่นก่อนที่จะอัพโหลด ให้แตะปุ่มเพิ่มเติม  บน iPhone หรือ iPad หรือคลิกแชร์บน Mac แล้วเลือกส่งสำเนา 
หากคุณแนบเอกสารจากแอพไฟล์บน iPhone หรือ iPad หรือแอพ Finder บน Mac การแก้ไขของคุณจะไม่ปรากฏในเอกสารนั้น 
หากผู้ที่ทำงานร่วมกับคุณลบวัตถุ, สไลด์ Keynote หรือชีต Numbers ที่คุณแก้ไขแบบออฟไลน์ไว้ การแก้ไขเหล่านั้นจะไม่ปรากฏในเอกสารเมื่อคุณกลับมาออนไลน์และซิงค์เอกสารด้วย Box
ดูตัวอย่าง แก้ไข และสร้างเอกสาร iWork ใน Box แบบออนไลน์
คุณสามารถดูตัวอย่าง แก้ไข และสร้างเอกสาร iWork ภายในเบราเซอร์เมื่อลงชื่อเข้าใช้บัญชี Box ของคุณ
3 คุณสามารถทำสิ่งต่อไปนี้ได้จากหน้าจอไฟล์และโฟลเดอร์
หากต้องการดูตัวอย่างเอกสาร iWork ทางออนไลน์ใน Box ให้เลือกเอกสารนั้น 
หากต้องการแก้ไขเอกสาร iWork ทางออนไลน์ใน Box ให้เลือกเอกสารนั้น เลือก Open (เปิด) ในแถบเครื่องมือ แล้วเลือก Pages, Numbers หรือ Keynote บนเว็บ
หากต้องการแก้ไขเอกสาร iWork ใหม่ทางออนไลน์ใน Box ให้คลิก New (ใหม่) 
แล้วเลือกเอกสาร Pages, สเปรดชีต Numbers หรืองานนำเสนอ Keynote
ดูเพิ่มเติม
ดูข้อมูลช่วยเหลือเกี่ยวกับ Pages, Numbers และ Keynote
ดูข้อมูลช่วยเหลือเกี่ยวกับ Box
ข้อกำหนดและความเป็นส่วนตัว
ระหว่างที่ใช้การทำงานร่วมกันบน iWork หรือดูการแสดงตัวอย่างหรือเอกสาร iWork บนเว็บ ไฟล์ของคุณจะจัดเก็บอยู่ในเซิร์ฟเวอร์ที่ปลอดภัยของ Apple ชั่วคราว
การใช้แอพพลิเคชั่น iWork ของคุณอยู่ภายใต้ข้อกำหนดในการให้บริการและนโยบายความเป็นส่วนตัวของ iWork บนเว็บของ Apple
หากคุณมีบัญชี Box ส่วนบุคคล คุณสามารถปิดการทำงานร่วมกันและการดูตัวอย่างบน Box ได้ในแท็บ Integrations (การรวม) ใน Account Settings (การตั้งค่าบัญชี) ของคุณ 
หากคุณมีบัญชี Box ขององค์กร โปรดติดต่อองค์กรของคุณ
1. หากต้องการใช้การทำงานร่วมกันบน iWork ผ่านบัญชี Box สำหรับองค์กร องค์กรของคุณจะต้องเปิดการทำงานร่วมกันบน iWork ก่อน
2. ตรวจสอบให้แน่ใจว่าคุณได้เพิ่ม Box ลงในตำแหน่งที่ตั้งของคุณบน iPhone หรือ iPad แล้ว 
เมื่อคุณเพิ่ม Box แล้ว คุณสามารถย้ายและคัดลอกเอกสารไปยัง Box ด้วยตัวจัดการเอกสารและแอพไฟล์ได้
3. คุณสามารถเปิดเอกสารที่มีขนาดถึง 1 GB ได้บน box.com 
หากเอกสารของคุณมีขนาดใหญ่เกิน 1 GB ให้ใช้ Pages, Numbers หรือ Keynote ใน iPhone, iPad หรือ Mac เพื่อลดขนาดก่อนที่จะลองเปิดใน box.com
