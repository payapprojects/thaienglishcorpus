ใช้หมายเลขโทรศัพท์มือถือของคุณเป็น Apple ID
เมื่อคุณสร้าง Apple ID ใหม่ในบางประเทศหรือภูมิภาค คุณสามารถใช้หมายเลขโทรศัพท์มือถือเป็นชื่อผู้ใช้ Apple ID ของคุณได้
วิธีการทำงานมีดังนี้
ทุกครั้งที่คุณตั้งค่า iPhone, iPad หรือ iPod touch คุณจะมีตัวเลือกในการลงชื่อเข้าด้วย Apple ID ที่มีอยู่หรือสร้างบัญชีใหม่ 
หากคุณสร้าง Apple ID ใหม่บน iPhone ที่ใช้ iOS 11 หรือใหม่กว่า คุณอาจจะสามารถใช้หมายเลขโทรศัพท์มือถือเป็นชื่อผู้ใช้ของคุณในจีนแผ่นดินใหญ่และอินเดีย 
หากคุณอาศัยอยู่ในจีนแผ่นดินใหญ่ ให้ใช้หมายเลขโทรศัพท์มือถือ (+86) ของคุณ 
หากคุณอาศัยอยู่ในอินเดีย ให้ใช้หมายเลข (+91) ของคุณ 
คุณอาจสร้างบัญชีใหม่ด้วยหมายเลขโทรศัพท์ในอุปกรณ์ Android, สมาร์ททีวี และอุปกรณ์สตรีมบางรุ่นได้ ทั้งนี้ขึ้นอยู่กับผู้ให้บริการของคุณ 
เพียงเลือกสร้าง Apple ID ของคุณจากหน้าจอลงชื่อเข้า Apple ID และทำตามคำแนะนำเพื่อสร้างบัญชีใหม่ของคุณ
เมื่อสร้างบัญชีของคุณแล้ว คุณสามารถใช้บัญชีนั้นเพื่อลงชื่อเข้าใช้อุปกรณ์หรือบริการใดๆ ของ Apple ได้ 
เมื่อใดก็ตามที่คุณลงชื่อเข้าในอุปกรณ์ แอพ หรือบริการใหม่ คุณจะต้องป้อนหมายเลขโทรศัพท์มือถือของคุณ รวมถึงรหัสประเทศและรหัสผ่าน 
โปรดตรวจสอบให้แน่ใจว่าคุณได้ใช้ Apple ID เดียวกันในทุกที่ที่คุณลงชื่อเข้า เพื่อให้อุปกรณ์และบริการของ Apple ของคุณทำงานร่วมกันได้อย่างราบรื่น
หากคุณมี Apple ID ที่เป็นที่อยู่อีเมลอยู่แล้ว คุณอาจไม่สามารถเปลี่ยนเป็นหมายเลขโทรศัพท์ได้ 
อย่างไรก็ตาม คุณสามารถผูกบัญชีของคุณกับหมายเลขโทรศัพท์หรือที่อยู่อีเมลอื่นในหน้าบัญชี Apple ID ของคุณได้
สิ่งที่คุณต้องมี
ต่อไปนี้คืออุปกรณ์ที่คุณต้องใช้หมายเลขโทรศัพท์มือถือเป็น Apple ID เพื่อลงชื่อเข้า
iPhone, iPad หรือ iPod touch ที่ใช้ iOS 11 หรือใหม่กว่า
Mac ที่ใช้ macOS High Sierra หรือใหม่กว่า
Apple Watch ที่ใช้ watchOS 4 หรือใหม่กว่า
Apple TV ที่ใช้ tvOS 11 หรือใหม่กว่า
Apple.com/th, iCloud.com และคุณสมบัติเว็บอื่นๆ ของ Apple*
บริการของ Apple บนอุปกรณ์ของผู้ให้บริการรายอื่น
*คุณอาจได้รับแจ้งให้เปลี่ยนชื่อผู้ใช้จากหมายเลขโทรศัพท์เป็นที่อยู่อีเมล ทั้งนี้ขึ้นอยู่กับวิธีที่คุณสร้าง Apple ID ของคุณ 
คุณอาจต้องให้ข้อมูลเพิ่มเติมด้วยเช่นกัน
ความปลอดภัยของบัญชี
ข้อกำหนดด้านความปลอดภัยของบัญชีอาจแตกต่างกันไป ขึ้นอยู่กับว่าคุณสร้างบัญชีของคุณที่ใด และเกี่ยวข้องกับข้อมูลส่วนบุคคลมากเท่าใด 
เมื่อใช้อุปกรณ์ Apple คุณควรใช้การตรวจสอบสิทธิ์สองปัจจัย 
ในบางกรณี คุณอาจได้รับแจ้งให้ตรวจสอบหรืออัพเกรดความปลอดภัยเมื่อลงชื่อเข้าใช้บนอุปกรณ์ของ Apple หรือเว็บไซต์ของ Apple
ข้อความจาก Apple
เนื่องจาก Apple ID ที่ใช้หมายเลขโทรศัพท์ไม่ได้เชื่อมโยงกับที่อยู่อีเมลใด Apple จึงสื่อสารกับคุณผ่านทางแอพข้อความบน iPhone, iPad หรือ iPod touch และหน้าบัญชี Apple ID ของคุณ
เพื่อให้แน่ใจว่าคุณจะได้รับข้อความสำคัญที่เกี่ยวข้องกับบัญชีผ่านทาง iPhone, iPad หรือ iPod touch ของคุณ ให้ลงชื่อเข้าใช้ iCloud, iMessage และ FaceTime ด้วย Apple ID ที่เป็นหมายเลขโทรศัพท์ของคุณ 
หากคุณลงชื่อเข้าใช้ iCloud บน iPhone, iPad หรือ iPod touch โดยที่ไม่มีซิมการ์ด คุณจะไม่สามารถใช้ FaceTime และ iMessage ได้ 
คุณสามารถดูข้อความจาก Apple ได้ตลอดเวลาที่หน้าบัญชี Apple ID
เปลี่ยน Apple ID ของคุณเป็นหมายเลขโทรศัพท์มือถืออื่น
คุณอาจสามารถเปลี่ยน Apple ID เป็นหมายเลขโทรศัพท์มือถืออื่นจากหน้าบัญชี Apple ID หรือบน iPhone, iPad หรือ iPod touch ของคุณได้ ทั้งนี้ขึ้นอยู่กับประเทศหรือภูมิภาคที่คุณอาศัยอยู่และวิธีที่คุณสร้าง Apple ID ของคุณ
ในจีนแผ่นดินใหญ่ Apple ID ใหม่ของคุณจะต้องเป็นหมายเลขโทรศัพท์มือถือ (+86) หมายเลขอื่น 
ในอินเดีย Apple ID ใหม่ของคุณต้องเป็นหมายเลขโทรศัพท์มือถือ (+91) หมายเลขอื่น

หากหมายเลขโทรศัพท์มือถือใหม่ของคุณเชื่อมต่อกับ Apple ID แล้ว โดยใช้เป็นหมายเลขโทรศัพท์ iMessage หมายเลขโทรศัพท์ที่แชร์ หรือหมายเลขโทรศัพท์ที่เชื่อถือได้ คุณจะไม่จำเป็นต้องตรวจสอบยืนยันหมายเลขนั้น
ไม่ได้รับรหัสใช่ไหม หากคุณใช้ iOS 11.3 หรือใหม่กว่าใน iPhone คุณอาจไม่จำเป็นต้องป้อนรหัสการตรวจสอบยืนยัน
