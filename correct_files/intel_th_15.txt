ถ้าความเร็ว (หรือความเร็วที่ไม่เพียงพอ) ของคอมพิวเตอร์กำลังทำให้คุณรู้สึกเบื่อ นั่นอาจเป็นสัญญาณที่แสดงว่า RAM ของคอมพิวเตอร์อาจมีไม่เพียงพอที่จะตอบสนองความต้องการของคุณได้ 
หากคอมพิวเตอร์ของคุณมีอายุอยู่ระหว่างสองถึงสี่ปี นั่นอาจหมายความว่าโปรเซสเซอร์ของคุณช้าเกินกว่าที่จะรับมือกับความต้องการของแอพพลิเคชั่นที่ทรงพลังขึ้นในปัจจุบัน
นี่อาจเป็นเวลาที่ควรเริ่มคิดถึงการอัพเกรดคอมพิวเตอร์
โดยคุณสามารถคลิกไปที่Intel Product Finderเพื่อค้นหาโน้ตบุ๊กที่เหมาะสมกับคุณที่สุดได้
RAM ของคอมพิวเตอร์และโน้ตบุ๊กทำงานอย่างไร
RAM ย่อมาจาก Random-Access Memory (หน่วยความจำเข้าถึงโดยสุ่ม) แล้วหมายความว่าอย่างไร โดยพื้นฐานแล้ว RAM ของคอมพิวเตอร์ คือหน่วยความจำระยะสั้นสำหรับจัดเก็บข้อมูลที่โปรเซสเซอร์ต้องการใช้
ซึ่งจะต่างจากข้อมูลระยะยาวที่จัดเก็บไว้ในฮาร์ดไดรฟ์ของคุณ และจะยังคงอยู่แม้ว่าจะปิดคอมพิวเตอร์ไปแล้ว
เมื่อใดก็ตามที่คุณเล่นเกมจากฮาร์ดไดรฟ์ของคอมพิวเตอร์หรือสตรีมภาพยนตร์จากอินเทอร์เน็ต ข้อมูลทั้งหมดที่โปรเซสเซอร์ของคอมพิวเตอร์ต้องการใช้ในการเล่นเกมหรือรับชมภาพยนตร์จะถูกจัดเก็บไว้ใน RAM 
เพื่อให้โปรเซสเซอร์ของคุณสามารถเข้าถึงข้อมูลเหล่านั้นได้อย่างรวดเร็ว
เมื่อคุณเล่นเกมหรือชมภาพยนตร์เสร็จและทำการปิดแอพพลิเคชันดังกล่าวแล้ว โปรเซสเซอร์จะไม่ต้องการใช้ข้อมูลนั้นอีก ข้อมูลดังกล่าวจะถูกแทนที่ด้วยงานถัดไปที่คุณทำ 
RAM อาจทำให้คอมพิวเตอร์ของคุณทำงานช้าได้หากมีจำนวนไม่เพียงพอสอดคล้องกับงานที่คุณร้องขอให้โปรเซสเซอร์ดำเนินการ
ถ้าโปรเซสเซอร์ต้องการโหลดข้อมูลมากกว่าที่ RAM ของคอมพิวเตอร์จะสามารถจัดการได้ โปรเซสเซอร์จะต้องกลับเข้าไปยังฮาร์ดไดรฟ์หรืออินเทอร์เน็ตเพื่อรับข้อมูลอีกครั้ง
เปรียบเหมือนการพยายามเก็บลูกเทนนิสให้มากกว่าจำนวนที่สามารถขนได้ และท้ายที่สุดก็จะใช้เวลาเก็บลูกเทนนิสมากกว่าการเล่นเทนนิสเสียอีก!
โปรเซสเซอร์ของคอมพิวเตอร์ทำงานอย่างไร
โปรเซสเซอร์คือส่วนประกอบที่ทำให้คอมพิวเตอร์สามารถคิดได้ เปรียบเสมือนส่วนสติสัมปชัญญะของสมองของคุณ
ยิ่งสมองของคุณมีประสิทธิภาพมากเท่าใด คุณก็จะสามารถตอบปัญหาคณิตศาสตร์ อ่านและเข้าใจคำศัพท์ในหนังสือ เข้าใจในมุขตลก หรือสั่งร่างกายให้ทำในสิ่งที่ต้องการขณะเล่นกีฬาได้อย่างถูกต้องรวดเร็วยิ่งขึ้น 
โปรเซสเซอร์ของคอมพิวเตอร์ก็เช่นเดียวกัน
ยิ่งโปรเซสเซอร์มีประสิทธิภาพมากเท่าใด ก็จะทำให้โปรเซสเซอร์สามารถทำงานร่วมกับข้อมูล (เกม ภาพยนตร์ แอพพลิเคชั่น และอื่นๆ) 
ที่อยู่ในฮาร์ดไดรฟ์ของคุณและโลกออนไลน์ได้เร็วยิ่งขึ้น
ทั้งนี้ โปรเซสเซอร์ของคอมพิวเตอร์จะทำงานเป็นทีมร่วมกับ RAM 
RAM ก็เหมือนกับความจำระยะสั้นของคุณ 
ถ้าศีรษะของคุณโดนกระแทกและสูญเสียความจำระยะสั้นไป (RAM ของคุณ) คุณก็จะไม่สามารถจดจำสิ่งใดๆ ที่เกิดขึ้นนานเกินกว่าสองสามวินาทีที่แล้วได้
อย่างไรก็ตาม ความคิดอ่านของคุณจะยังคงใช้การได้อยู่ 
แต่ลองคิดดูว่าหากไม่มีความจำใดๆ คุณ (และโปรเซสเซอร์ของคุณ) จะพบกับความยุ่งยากแสนสาหัสเพียงใด
ประสิทธิภาพมากขึ้นขณะราคาลดลง
สิ่งที่คุณจะพบในตลาดปัจจุบันก็คือความสามารถของพีซีแบบ All-in-One โน้ตบุ๊กและอุปกรณ์ 2 in 1ในการนำเสนอประสิทธิภาพชั้นยอดได้อย่างคล่องแคล่วโดยที่เสียเงินน้อยกว่าที่คุณคิด
ถ้าคุณยังตัดสินใจเลือกซื้อคอมพิวเตอร์เครื่องใหม่ไม่ได้ ให้ลองพิจารณาคอมพิวเตอร์ประสิทธิภาพสูงบางรุ่นที่มีวางจำหน่ายในปัจจุบัน
แล้วคุณจะรู้สึกประหลาดใจกับสิ่งที่ได้พบ
