ดูข้อมูลช่วยเหลือในการใช้ Outlook กับ iCloud สำหรับ Windows

ลองทำตามขั้นตอนต่อไปนี้ หากคุณต้องการความช่วยเหลือเกี่ยวกับเมล iCloud, รายชื่อ และปฏิทินใน Outlook และดูเกี่ยวกับคุณสมบัติต่างๆ ของ iCloud ซึ่งจะทำงานแตกต่างกันระหว่างอุปกรณ์ Apple กับ Outlook

ก่อนที่คุณจะเริ่ม: 

  • ตรวจสอบให้แน่ใจว่า PC ของคุณตรงตามความต้องการของระบบสำหรับ iCloud
  • ดาวน์โหลด iCloud สำหรับ Windows
  • ลงชื่อเข้าใช้ iCloud สำหรับ Windows ด้วย Apple ID ของคุณ หากคุณไม่ได้ใช้ iCloud สำหรับ Windows ให้ตั้งค่าการตรวจสอบสิทธิ์สองปัจจัย และลงชื่อเข้าใช้ Outlook ด้วยชื่อผู้ใช้ของ Apple ID และรหัสผ่านเฉพาะแอพ
  • ตรวจสอบหน้าสถานะของระบบเพื่อดูว่ามีปัญหาที่ทราบใดๆ ที่ส่งผลกระทบต่อเมล iCloud, รายชื่อ หรือปฏิทินหรือไม่
  • ตรวจสอบให้แน่ใจว่าการตั้งค่าวันที่และเวลาบนอุปกรณ์ของคุณถูกต้อง
  • ทำสำเนาข้อมูล iCloud

หากคุณไม่มีอุปกรณ์ที่สามารถอัพเดทเป็น iOS 9 หรือ OS X El Capitan ได้ คุณสามารถตั้งค่าการตรวจสอบยืนยันสองขั้นตอนแทนได้

แก้ไขรายการที่ซ้ำกัน

  • หากคุณกำลังประสบปัญหาเกี่ยวกับปฏิทินที่ซ้ำกัน เราสามารถช่วยคุณลบออกได้
  • และหากคุณเห็นรายชื่อที่ซ้ำกัน เราก็ช่วยคุณลบออกได้เช่นกัน

ตรวจสอบการเชื่อมต่ออินเทอร์เน็ตของคุณ

  1. ลองเปิดapple.com/th/ หากคุณไม่สามารถเชื่อมต่อได้ ปฏิทิน รายชื่อ หรือเตือนความจำบน iCloud ก็จะไม่อัพเดทใน Outlook คุณต้องดูข้อมูลช่วยเหลือเกี่ยวกับการเชื่อมต่ออินเทอร์เน็ต
  2. ลองเปิดเว็บไซต์ที่ปลอดภัยจาก PC ของคุณ หากคุณเห็นข้อความแสดงข้อผิดพลาด เช่น "หมดเวลาการเชื่อมต่อ" คุณต้องติดต่อผู้ให้บริการอินเทอร์เน็ตเพื่อขอความช่วยเหลือ

ตรวจสอบการตั้งค่าการเข้าสู่ระบบ

หากคุณมี Apple ID หลายบัญชีที่คุณใช้กับ iCloud คุณอาจต้องเข้าสู่ระบบ iCloud สำหรับ Windows ด้วยบัญชีอื่นที่ไม่ใช่บัญชีที่อยู่ในอุปกรณ์อื่นๆ ของคุณ หากต้องการตรวจสอบ ให้เปิด iCloud สำหรับ Windows และเปรียบเทียบกับอุปกรณ์อื่นๆ ของคุณ

รีเฟรช Outlook

หากเมื่อเร็วๆ นี้ คุณได้ทำการเปลี่ยนแปลงใน Outlook โดยไม่มีการอัพเดทในอุปกรณ์อื่นๆ ของคุณ หรือทำการเปลี่ยนแปลงในอุปกรณ์อื่นๆ ของคุณโดยไม่มีการอัพเดทใน Outlook ให้คลิกรีเฟรชใน Outlook 

หากระบบไม่รองรับ Outlook เวอร์ชั่นที่ติดตั้ง

หากคุณได้รับข้อผิดพลาดที่แจ้งว่า iCloud ไม่รองรับ Outlook เวอร์ชั่นที่ติดตั้ง ให้ถอนการติดตั้ง Office เวอร์ชั่นที่คุณได้รับจาก Windows Store จากนั้นให้ทำตามขั้นตอนต่อไปนี้เพื่อติดตั้ง Office 365

หากตัวเลือกเมลในบัญชี iCloud สำหรับ Windows เป็นสีเทา

ก่อนใช้เมล iCloud คุณจะต้องสร้างที่อยู่อีเมล @icloud.com หากคุณไม่ได้ตั้งค่าบัญชีเมล iCloud เมื่อคุณตั้งค่า iCloud ตัวเลือกเมลในบัญชี iCloud สำหรับ Windows อาจเป็นสีเทา 

หากต้องการทำให้ตัวเลือกเมลใช้งานได้ ให้ตั้งค่าที่อยู่อีเมล @icloud.com จากนั้นปิดและเปิด iCloud สำหรับ Windows อีกครั้ง หากคุณยังไม่เห็นตัวเลือกเมล ให้ลงชื่อออก และกลับเข้าไปที่ iCloud สำหรับ Windows ใหม่

เมล

ดูข้อมูลช่วยเหลือเกี่ยวกับการตั้งค่าหรือการส่งเมล iCloud

หากคุณมีอีเมลที่อยู่ในกล่องออก

หากคุณเห็นอีเมลพักอยู่ในกล่องขาออก ให้ทำตามขั้นตอนเหล่านี้

  1. ย้ายข้อความจากกล่องออกไปยังโฟลเดอร์แบบร่าง หรือลบทิ้ง
  2. เปิดข้อความในโฟลเดอร์แบบร่างหรือรายการที่ถูกลบ แล้วคลิกส่งอีกครั้ง

หากคุณไม่สามารถตั้งค่าเมลโดยใช้ iCloud สำหรับ Windows ได้

คุณจำเป็นต้องตั้งค่าที่อยู่อีเมล @icloud.com ก่อนจึงจะสามารถใช้เมล iCloud ได้ หากคุณไม่ได้ตั้งค่าบัญชีเมล iCloud ในเวลาเดียวกันกับ iCloud ตัวเลือกเมลในบัญชี iCloud สำหรับ Windows อาจเป็นสีเทา หลังจากที่คุณตั้งค่าที่อยู่อีเมล @icloud.com แล้ว คุณสามารถทำให้ตัวเลือกเมลใช้ได้ เมื่อคุณปิดและเปิด iCloud สำหรับ Windows อีกครั้ง หากคุณยังไม่เห็นตัวเลือกเมล ให้ลงชื่อออก และกลับเข้าไปที่ iCloud สำหรับ Windows ใหม่

หากคุณไม่สามารถส่งหรือรับอีเมล

หากคุณไม่สามารถส่งหรือรับข้อความด้วยที่อยู่อีเมล iCloud ของคุณ ให้ทำตามขั้นตอนเหล่านี้

  1. ตรวจสอบว่าคุณสามารถส่งและรับอีเมลจาก iCloud.com และอุปกรณ์ iOS ของคุณได้
  2. ปิด Outlook
  3. ลงชื่อออกจาก iCloud สำหรับ Windows แล้วลงชื่อเข้าใช้ใหม่
  4. รีสตาร์ท Outlook

หากคุณยังคงส่งหรือรับเมลไม่ได้ คุณสามารถขอความช่วยเหลือเพิ่มเติมเกี่ยวกับ Outlook หรือติดต่อเราได้ 

หากเมลไม่ซิงค์ระหว่างอุปกรณ์

หากคุณสังเกตเห็นว่าคุณต้องลบอีเมลออกจากกล่องขาเข้าในอุปกรณ์ทุกเครื่อง แสดงว่าเมลของคุณอาจไม่ได้รับการซิงค์ระหว่างอุปกรณ์ iOS และ iCloud สำหรับ Windows หากต้องการซิงค์อุปกรณ์ของคุณ ให้ทำตามขั้นตอนเหล่านี้

  1. เปิด iCloud สำหรับ Windows
  2. ยกเลิกการเลือกเมล
  3. คลิกนำไปใช้
  4. เลือกเมลอีกครั้ง
  5. คลิกนำไปใช้
  6. ลงชื่อออกจาก iCloud สำหรับ Windows แล้วลงชื่อเข้าใช้ใหม่

หากอุปกรณ์ยังไม่ซิงค์อยู่ คุณสามารถดูข้อมูลเพิ่มเติม 

หากคุณต้องการให้ iCloud เป็นบัญชีเริ่มต้นสำหรับการส่งเมล

คุณสามารถเปลี่ยนการตั้งค่า Outlook เพื่อให้อีเมลถูกส่งโดยอัตโนมัติจากบัญชี iCloud ของคุณ หากต้องการทำให้ iCloud เป็นบัญชีเริ่มต้น ให้ทำตามขั้นตอนเหล่านี้ใน Outlook 2010 หรือใหม่กว่า

  1. ไปที่ไฟล์> การตั้งค่าบัญชี
  2. ที่แท็บข้อมูล ให้คลิกการตั้งค่าบัญชี
  3. เลือกบัญชี iCloud ของคุณจากรายการ แล้วคลิกตั้งเป็นค่าเริ่มต้น หากคุณไม่สามารถคลิกตั้งเป็นค่าเริ่มต้น แสดงว่าบัญชี iCloud ของคุณเป็นบัญชีเริ่มต้นอยู่แล้ว

ปฏิทิน รายชื่อ และเตือนความจำ

หากคุณต้องการความช่วยเหลือเกี่ยวกับปฏิทิน รายชื่อ และ เตือนความจำ ที่ใช้ iCloud สำหรับ Windows ให้ลองทำตามขั้นตอนเหล่านี้ 

เมื่อคุณเปิดเตือนความจำ iCloud บน PC ที่ใช้ Windows แอพนี้จะติดป้ายกำกับว่า "งาน" และปรากฏอยู่ในส่วน "งาน"

ปิดปฏิทิน iCloud, รายชื่อ และ เตือนความจำ แล้วเปิดใหม่

  1. ปิด Outlook
  2. เปิด iCloud สำหรับ Windows
  3. ยกเลิกการเลือกช่องทำเครื่องหมาย เมล รายชื่อ ปฏิทินและงาน และคลิกปรับใช้
  4. รอสักสองสามวินาที เลือกเมล รายชื่อ ปฏิทินและงาน แล้วคลิกปรับใช้
  5. เปิด Outlook

ตรวจสอบให้แน่ใจว่า Add-in ของ iCloud Outlook เปิดอยู่

Add-in ของ iCloud Outlook ทำหน้าที่ถ่ายโอนคอนเทนต์ระหว่าง iCloud กับ Outlook ทำตามขั้นตอนเหล่านี้ใน Outlook 2010 หรือใหม่กว่าเพื่อตรวจสอบและเปิด Add-in

  1. เลือกเมนูไฟล์
  2. คลิกตัวเลือกในแผงด้านซ้าย
  3. คลิก Add-in ในแผงด้านซ้ายของหน้าต่างตัวเลือก Outlook
  4. ดูรายชื่อ Add-in ในส่วน "Add-in ของแอปพลิเคชั่นที่ใช้งานอยู่" หาก iCloud Add-in แสดงอยู่ใต้โปรแกรมเสริมที่ไม่ใช้งานหรือถูกปิดใช้งาน โปรดตรวจสอบให้แน่ใจว่าได้เลือกโปรแกรมเสริม Com ในเมนูดรอปดาวน์ซึ่งอยู่ถัดจากจัดการที่ด้านล่างของหน้าต่างแล้ว จากนั้นคลิกไป
  5. เลือกกล่องกาเครื่องหมาย Add-in ของ iCloud Outlook จากนั้นคลิกตกลง
  6. รีสตาร์ท Outlook

ตรวจสอบให้แน่ใจว่า iCloud ไม่ได้เป็นบัญชีเริ่มต้นของคุณในไฟล์ข้อมูล

เมื่อมีการตั้งค่า iCloud เป็นบัญชีเริ่มต้นของคุณในไฟล์ข้อมูลแล้ว ข้อมูลของคุณก็จะไม่ซิงค์ระหว่างอุปกรณ์ 

  1. เลือกไฟล์> ข้อมูล> การตั้งค่าบัญชี > ไฟล์ข้อมูล 
  2. หาก iCloud ถูกระบุว่าเป็นบัญชีเริ่มต้นในคอลัมน์ข้อคิดเห็น ให้เลือกบัญชีอื่น
  3. คลิกตั้งเป็นค่าเริ่มต้น

รีสตาร์ทคอมพิวเตอร์ของคุณ

ปิดคอมพิวเตอร์ แล้วเปิดใหม่

คุณสมบัติของ iCloud ที่จะทำงานแตกต่างกันใน Outlook

คุณสมบัติบางอย่างของ iCloud ใน Outlook จะทำงานแตกต่างจากที่ทำในอุปกรณ์ Apple ของคุณ ต่อไปนี้คือเคล็ดลับบางประการที่ช่วยให้คุณใช้ Outlook กับ iCloud สำหรับ Windows 

เมลและโน้ต

  • โฟลเดอร์ผู้ส่ง VIP จะไม่สามารถใช้ได้ใน Outlook
  • กฎของเมลใดๆ ที่คุณสร้างใน Outlook จะไม่สามารถใช้ได้ในแอพพลิเคชั่นเมลสำหรับอุปกรณ์อื่นๆ ของคุณ หรือบน iCloud.com/
  • ลายเซ็นจะถูกบันทึกในเครื่องลงในอุปกรณ์ของคุณ คุณจะไม่เห็นลายเซ็นที่คุณสร้างใน Outlook บนอุปกรณ์อื่นๆ ของคุณ

โน้ตไม่ได้เป็นแอพพลิเคชั่นแบบแยกต่างหาก โน้ตของคุณจะถูกเก็บไว้ในโฟลเดอร์เมล 

  • คุณจะพบโน้ตที่คุณสร้างขึ้นบนอุปกรณ์อื่นๆ ในโฟลเดอร์นี้
  • โน้ตที่คุณโพสต์จาก Outlook ไปยังโฟลเดอร์นี้จะไม่ปรากฏบนอุปกรณ์อื่นๆ ของคุณ แต่จะเห็นได้บนคอมพิวเตอร์เครื่องอื่นๆ ที่ใช้ Outlook และ iCloud สำหรับ Windows เมื่อคุณลงชื่อเข้าใช้ด้วย Apple ID เดียวกัน
  • คุณสามารถลบโน้ตจากโฟลเดอร์นี้ และโน้ตจะถูกลบออกจากอุปกรณ์อื่นๆ ของคุณด้วย

รายชื่อ

  • Outlook จะนำเข้ารายชื่อแรกในชุด vCard เท่านั้น 
  • หากคุณจำเป็นต้องเพิ่มชุด vCard ไปยัง Outlook ให้นำเข้าชุดดังกล่าวไปยัง iCloud.com หรือรายชื่อ macOS โดยตรง เราจะรวมทั้งชุดไว้ให้คุณ และเพิ่มเป็นบุคคลในรายชื่อของคุณ 

ปฏิทิน

  • เมื่อคุณเพิ่มไฟล์แนบปฏิทินไปยังรายการปฏิทินใน Outlook แล้ว คุณจะไม่เห็นไฟล์แนบนั้นในปฏิทินสำหรับ iOS หรือปฏิทินสำหรับ macOS (หรือในทางกลับกัน)

เตือนความจำ (งาน)

คุณต้องป้อนเตือนความจำลงในส่วน "งานใน Outlook" ของ iCloud เช่นเดียวกับปฏิทิน และรายชื่อ หากคุณไม่ได้ป้อนงานลงในส่วน iCloud งานนั้นก็จะไม่ถ่ายโอนไปยัง iCloud

การให้ข้อมูลเกี่ยวกับผลิตภัณฑ์ที่ไม่ได้ผลิตโดย Apple หรือเว็บไซต์อิสระที่ Apple ไม่ได้ควบคุมหรือทดสอบไม่ถือเป็นการแนะนำหรือการรับรองใดๆ Apple จะไม่รับผิดชอบในส่วนที่เกี่ยวข้องกับการเลือก ประสิทธิภาพการทำงาน หรือการใช้งานเว็บไซต์หรือผลิตภัณฑ์ของบริษัทอื่น Apple ไม่รับรองความถูกต้องหรือความน่าเชื่อถือของเว็บไซต์ของบริษัทอื่น โปรดติดต่อผู้จำหน่ายหากต้องการข้อมูลเพิ่มเติม
