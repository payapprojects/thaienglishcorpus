การถ่ายโอนไฟล์จากพีซีเครื่องหนึ่งไปยังอีกเครื่องหนึ่งสามารถทำได้อย่างรวดเร็วและปลอดภัย โดยไม่มีข้อมูลสูญหายระหว่างกระบวนการ

โหลดข้อมูลของคุณไปยังพีซีเครื่องใหม่

  1. 1

    สำรองข้อมูลจากพีซีเครื่องเก่าของคุณ

  2. 2

    เลือกวิธีถ่ายโอนไฟล์ที่สะดวกที่สุดสำหรับคุณ

  3. 3

    รับเครื่องมือหรือบริการที่จำเป็นสำหรับการถ่ายโอนไฟล์ของคุณ หรือตั้งค่าเครือข่ายในบ้าน

  4. 4

    เก็บคอมพิวเตอร์เครื่องเก่าของคุณไว้สองถึงสามสัปดาห์ และตรวจสอบให้แน่ใจว่าได้ล้างข้อมูลในเครื่องแล้วก่อนนำไปทิ้ง

ไม่มีอะไรจะน่าตื่นเต้นไปกว่าการได้ใช้คอมพิวเตอร์เครื่องใหม่อันทรงพลังที่ทำให้คุณสามารถทำงาน ใช้งานแอพพลิเคชั่นใหม่ๆ และเล่นเกมได้ด้วยความเร็วที่น่าทึ่ง อย่างไรก็ตาม ไม่มีอะไรจะน่ากังวลใจมากไปกว่าการที่คุณพบว่าจะต้องย้ายเพลง รูปภาพ เอกสาร และไฟล์อื่นๆ บนคอมพิวเตอร์เครื่องเก่าของคุณไปยังคอมพิวเตอร์เครื่องใหม่

แต่โชคดีที่คุณไม่จำเป็นต้องอ้อนวอนหรือจ้างเพื่อนของคุณเพื่อขอให้ช่วยเหมือนกับที่ย้ายบ้าน เพราะว่ามีหนทางมากมายในการแชร์ไฟล์อย่างง่ายดายและปลอดภัย หรือแม้กระทั่งย้ายโปรแกรมต่างๆ จากคอมพิวเตอร์เครื่องเก่าของคุณไปยังเครื่องใหม่ ในขณะที่ยังรักษาข้อมูลของคุณให้ปลอดภัย

วิธีการแบบ DIY: ถ่ายโอนข้อมูลด้วยตนเอง
หากคุณเป็นคนที่ชอบลงมือทำเอง หนึ่งในวิธีที่ง่ายที่สุดในการย้ายไฟล์ทั้งหมดของคุณเข้าสู่คอมพิวเตอร์เครื่องใหม่ก็คือ การย้ายด้วยตนเอง ซึ่งสามารถทำได้สองวิธี

สำหรับมือใหม่ ให้เชื่อมต่อคอมพิวเตอร์ของคุณเข้ากับแฟลชไดรฟ์ USB หรือฮาร์ดไดรฟ์ภายนอก ซึ่งอาจมีราคาตั้งแต่ 30 ดอลลาร์สหรัฐฯ ไปจนถึงหลายร้อยดอลลาร์ ขึ้นอยู่กับปริมาณของพื้นที่จัดเก็บ ความเร็ว และฟังก์ชันการทำงานที่คุณต้องการ เพียงแค่คัดลอกไฟล์ของคุณลงในไดรฟ์ภายนอก แล้วถอดอุปกรณ์เก็บข้อมูลแล้วเสียบเข้ากับคอมพิวเตอร์เครื่องใหม่ จากนั้นให้ทำขั้นตอนย้อนกลับเพื่อโหลดไฟล์ลงในเครื่อง เคล็ดลับ: คอมพิวเตอร์บางเครื่องมีพอร์ต eSATA ที่ออกแบบมาเป็นพิเศษสำหรับฮาร์ดไดรฟ์ภายนอก และสามารถย้ายข้อมูลได้รวดเร็วกว่าพอร์ต USB

หากคุณไม่ต้องการซื้อฮาร์ดไดรฟ์ภายนอก คุณก็สามารถย้ายและจัดเก็บไฟล์ของคุณลงในคลาวด์ได้ โดยใช้บริการจัดเก็บข้อมูลบนอินเทอร์เน็ต เช่น Microsoft OneDrive*, Google Drive* หรือ Dropbox* ส่วนวิธีการนั้นก็เหมือนกับก่อนหน้า โดยเพียงแค่ลากและปล่อยไฟล์จากคอมพิวเตอร์เครื่องเก่าของคุณไปยังบริการคลาวด์ แล้วจากนั้นก็ลากและปล่อยไฟล์ลงบนคอมพิวเตอร์เครื่องใหม่ บริการเหล่านี้มักจะใช้งานได้ฟรีสำหรับที่เก็บข้อมูลขนาดเล็ก จากนั้นจะเรียกเก็บค่าบริการรายเดือนเมื่อความต้องการที่เก็บข้อมูลของคุณเพิ่มขึ้น

เพิ่มความเร็วให้กับทุกสิ่ง: ใช้สายถ่ายโอนข้อมูล
หากคุณพบว่าการใช้ที่เก็บข้อมูลบนคลาวด์เป็นเรื่องเสียเวลา หรือหากคุณต้องการหลีกเลี่ยงการสมัครใช้บริการรายเดือน อีกทางเลือกหนึ่งที่คุณมีก็คือ การย้ายไฟล์ของคุณด้วยสายถ่ายโอนข้อมูล สายนี้จะเสียบเข้ากับพอร์ตของทั้งคอมพิวเตอร์เครื่องใหม่และเครื่องเก่าของคุณ โดยทั่วไป สายเคเบิลจะมาพร้อมกับซอฟต์แวร์ที่จะทำการถ่ายโอนไฟล์โดยอัตโนมัติเมื่อทำการอัปเกรดจากคอมพิวเตอร์รุ่นเก่าที่ใช้ระบบ Windows® 10, 8.1, 8, 7, Vista, หรือ XP ไปยังคอมพิวเตอร์ที่ใหม่กว่า สายถ่ายโอนข้อมูลถ่ายโอนได้รวดเร็วกว่าฮาร์ดไดรฟ์ภายนอก เนื่องจากคอมพิวเตอร์เครื่องเก่าจะเชื่อมต่อโดยตรงกับเครื่องใหม่ พร้อมช่วยให้ไม่ต้องใช้สื่อกลาง (ไดรฟ์ภายนอก)

ก่อนที่คุณจะเริ่มถ่ายโอนทุกไฟล์ ให้พิจารณาก่อนว่าคุณต้องการข้อมูลทั้งหมดหรือไม่ การตั้งค่าคอมพิวเตอร์เครื่องใหม่ถือเป็นโอกาสที่ดีเยี่ยมที่จะทิ้งไฟล์เก่าๆ ด้วยการจัดเก็บถาวร เพื่อทำให้ไฟล์และโครงสร้างโฟลเดอร์ของคุณเป็นระเบียบ

Jill Duffy, PC Magazine

จ้างคนย้าย (พีซี): ใช้ซอฟต์แวร์อเนกประสงค์
ใน Windows เวอร์ชันเก่าจะมีบริการที่ชื่อว่า Easy Transfer* ซึ่งทำให้ผู้ใช้สามารถย้ายไฟล์ระหว่างคอมพิวเตอร์ได้อย่างง่ายดาย น่าเสียดายที่บริการนี้ถูกยกเลิกไปใน Windows® 10 แต่อย่ากังวลไปเพราะว่า Microsoft ได้ร่วมมือกับผู้ให้บริการ Laplink PC Mover* เพื่อเพิ่มฟังก์ชันการทำงานที่เหมือนกับ Easy Transfer PCMover Express* ของผู้ให้บริการจะถ่ายโอนข้อมูลและการตั้งค่าจากคอมพิวเตอร์ที่ใช้ Windows XP ไปยังคอมพิวเตอร์ที่ใช้ Windows 8.1 หรือใหม่กว่า บริการนี้มีราคาประมาณ 15 ดอลลาร์สหรัฐฯ PC Mover Professional มีราคาแพงกว่า แต่ก็ทำให้คุณสามารถย้ายแอพพลิเคชั่นของคุณได้ด้วย หากคุณมีปัญหา บริการทั้งสองมีบริการช่วยเหลือทุกวันตลอด 24 ชั่วโมง

คุณจำเป็นต้องเสียค่าใช้จ่ายหากใช้วิธีการนี้ แต่กระบวนนั้นก็เป็นไปโดยอัตโนมัติ และสามารถให้คำแนะนำกับคุณเพื่อให้แน่ใจว่าคุณจะไม่มองข้ามไฟล์ที่อาจอยู่ไว้ในตำแหน่งที่ไม่คุ้นเคยในคอมพิวเตอร์เครื่องเก่าของคุณ

ระบบปฏิบัติการที่หลากหลาย: ถ่ายโอนไฟล์จาก Mac ไปยังพีซี หรือจากพีซีไปยัง Mac
คุณสามารถถ่ายโอนไฟล์ด้วยตนเองระหว่าง Mac และพีซีได้ด้วยวิธีเดียวกับที่คุณย้ายไฟล์ระหว่างพีซีด้วยกัน ด้วยไดรฟ์ภายนอก บริการคลาวด์ หรือเครือข่ายในบ้าน หรือคุณจะให้คอมพิวเตอร์ทำหน้าที่เป็นผู้ช่วยส่วนตัวที่ดูแลการย้ายและการแชร์ไฟล์ได้ เนื่องจาก Mac OS X Lion และระบบปฏิบัติการรุ่นใหม่กว่ามีเครื่องมือเจ๋งๆ ที่ชื่อว่า Migration Assistant ซึ่งจะถ่ายโอนข้อมูลผู้ติดต่อ ปฏิทิน บัญชีอีเมล และอีกมากมายจากพีซี แล้ววางไฟล์เหล่านี้ในตำแหน่งที่เหมาะสมบน Mac เครื่องใหม่ของคุณ

ล้างข้อมูลให้เรียบ: เก็บข้อมูลเก่าของคุณไว้ให้พ้นจากผู้อื่น
หลังจากที่ย้ายไฟล์ของคุณแล้ว คุณควรเก็บคอมพิวเตอร์เครื่องเก่าของคุณอย่างน้อยสองถึงสามสัปดาห์ไว้ในกรณีที่คุณมองข้ามไฟล์บางส่วนไป อย่างไรก็ตาม หากคุณจะขาย ทิ้ง หรือให้คอมพิวเตอร์เครื่องเก่าของคุณกับผู้อื่น ควรตรวจสอบให้แน่ใจว่าได้นำข้อมูลทั้งหมดและข้อมูลส่วนตัวทั้งหมดของคุณออกไปแล้ว ซึ่งการถ่ายโอนไฟล์เพียงอย่างเดียวไม่สามารถทำได้ แม้แต่การลบไฟล์หรือฟอร์แมตดิสก์ก็อาจยังคงไม่ปลอดภัยจากมืออาชีพที่สามารถเรียกคืนข้อมูลที่คุณไม่ต้องการให้พวกเขาครอบครอง เช่น ข้อมูลธนาคาร หรืออีเมลเก่าๆ โชคดีที่โปรแกรมทำลายข้อมูลฟรี เช่น Darik's Boot And Nuke (DBAN) สามารถลบทุกสิ่งในคอมพิวเตอร์เครื่องเก่าของคุณได้ เพื่อให้มั่นใจว่าคุณจะไม่ทิ้งร่องรอย (ข้อมูล) ไว้ในคอมพิวเตอร์เครื่องเก่า

ปกป้องอยู่เสมอ: สำรองข้อมูลใหม่ๆ
เมื่อคุณคุ้นเคยกับการย้ายไฟล์ออกจากพีซีเครื่องเก่าแล้ว ก็เป็นโอกาสดีที่จะตรวจสอบให้แน่ใจว่าคุณมีระบบสำรองข้อมูลที่ใช้งานอยู่ เพื่อรองรับกรณีที่คุณจำเป็นต้องเรียกคืนไฟล์เหล่านั้นอีกครั้ง Windows* 8 และรุ่นใหม่กว่ามีคุณสมบัติที่มีประโยชน์ที่เรียกว่า File History ซึ่งทำให้คุณสามารถสำรองไฟล์ไปยังฮาร์ดไดรฟ์ภายนอกได้โดยอัตโนมัติและเป็นประจำ คุณสมบัตินี้ช่วยยืนยันว่าข้อมูลของคุณจะไม่หายไป หากมีเหตุการณ์ร้ายแรงเกิดขึ้น

หากคุณมีเครื่อง Mac คุณจะมีทางเลือกและแพลตฟอร์มที่หลากหลายสำหรับการสำรองข้อมูลของคุณ ไม่ว่าจะเป็น iCloud ไปจนถึงโปรแกรมอย่าง Time Machine ขึ้นอยู่กับเวอร์ชันของ OS X ที่คุณใช้

ข้อควรจำ: ถึงแม้ว่าจะมีวิธีง่ายๆ มากมายในการย้ายและแชร์ข้อมูล แต่คุณสามารถย้ายได้เฉพาะข้อมูลที่คุณเข้าถึงได้ จึงควรตรวจสอบให้มั่นใจว่าเอกสาร รูปภาพ และข้อมูลที่มีค่าอื่นๆ ของคุณได้รับการปกป้องอยู่เสมอ แม้ว่าคอมพิวเตอร์ของคุณจะสูญหายหรือเสียหาย ซึ่งคุณสามารถทำได้โดยอัตโนมัติอย่างง่ายดาย และช่วยยืนยันว่าการย้าย (คอมพิวเตอร์) ครั้งต่อไปของคุณจะราบรื่นขึ้น

เครื่อง 2-in-1 ที่ใช้โปรเซสเซอร์ Intel® Core™ รวมสมรรถนะของแล็ปท็อปเข้ากับความคล่องตัวสุดขีดของอุปกรณ์ที่บางเฉียบทันสมัย เนื่องจากมีดีไซน์ที่มีสไตล์และนำสมัยกับขนาดหน้าจอหลากหลาย จึงมี 2-in-1 ที่ขับเคลื่อนด้วย Intel® ที่ตรงกับทุกไลฟ์สไตล์

All-in-One

พีซีแบบ All-in-One ที่ใช้โปรเซสเซอร์ Intel® รวมข้อดีของอุปกรณ์ต่างๆ ไว้ในคอมพิวเตอร์ประสิทธิภาพสูงเครื่องเดียวที่มากประโยชน์ใช้สอย ปรับปรุงระบบเดสก์ท็อปทั้งระบบให้เป็นอุปกรณ์ที่สง่างามและทรงพลัง

เดสก์ท็อป

เดสก์ท็อปที่ขับเคลื่อนด้วย Intel® มาในรูปแบบของแพ็คเกจมากมาย ตั้งแต่อุปกรณ์ขนาดพกพาไปจนถึงทาวเวอร์ประสิทธิภาพสูง เพื่อตอบสนองความต้องการทุกรูปแบบ เดสก์ท็อปรุ่นใหม่ล่าสุดทำให้คุณมีอิสรภาพในการจัดเก็บข้อมูล สร้างสรรค์ และแบ่งปันเนื้อหาที่มีค่าที่สุดของคุณ

แล็ปท็อป

คว้าโอกาสฉบับพกพาด้วยแล็ปท็อปที่ใช้โปรเซสเซอร์ Intel® Core™ รุ่นใหม่ มาพร้อมพลังประมวลผลและความเร็วในการตอบสนองอย่างที่ไม่เคยมีมาก่อน จะทำให้คุณเปิดประตูสู่ก้าวใหม่ของประสิทธิภาพการทำงาน ความคิดสร้างสรรค์ และความบันเทิง
