วิธีลบดิสก์สำหรับ Mac

ใช้ยูทิลิตี้ดิสก์เพื่อลบ (ฟอร์แมต) ฮาร์ดดิสก์ SSD แฟลชไดรฟ์ หรืออุปกรณ์จัดเก็บข้อมูลอื่นๆ

การลบดิสก์ของคุณ: สำหรับเหตุผลที่ควรลบโดยส่วนใหญ่ รวมถึงเมื่อทำการฟอร์แมตดิสก์หรือขาย ยกให้ผู้อื่น หรือนำ Mac ของคุณมาแลก คุณควรลบข้อมูลดิสก์ทั้งหมดของคุณ

การลบดิสก์โวลุ่มบนดิสก์ของคุณ: ในกรณีอื่นๆ เช่น เมื่อดิสก์ของคุณมีดิสก์โวลุ่มหลายรายการ (หรือพาร์ติชั่น) และคุณไม่ต้องการที่จะลบทั้งหมด คุณสามารถลบดิสก์โวลุ่มที่เฉพาะเจาะจงบนดิสก์ได้

การลบดิสก์หรือโวลุ่มจะลบไฟล์ทั้งหมดถาวร ก่อนที่จะดำเนินการต่อ โปรดตรวจสอบให้แน่ใจว่าคุณได้สำรองไฟล์ใดๆ ที่คุณต้องการเก็บไว้แล้ว

วิธีการลบดิสก์ของคุณ

  1. เริ่มต้นระบบจากการกู้คืน macOS จากนั้นเลือกยูทิลิตี้ดิสก์จากหน้าต่างยูทิลิตี้ แล้วคลิกดำเนินการต่อ
    หากคุณไม่ได้กำลังลบดิสก์ที่ Mac ของคุณใช้เริ่มต้นระบบ คุณไม่จำเป็นต้องเริ่มต้นระบบจากการกู้คืน macOS แต่เพียงแค่เปิดยูทิลิตี้ดิสก์จากโฟลเดอร์ยูทิลิตี้ในโฟลเดอร์แอพพลิเคชั่นของคุณ 
  2. เลือกมุมมอง > แสดงอุปกรณ์ทั้งหมด จากแถบเมนูในยูทิลิตี้ดิสก์ แล้วแถบด้านข้างจะแสดงดิสก์ของคุณ (อุปกรณ์) รวมถึงตัวบรรจุและดิสก์โวลุ่มทั้งหมดภายในนั้น ดิสก์ที่ Mac ของคุณใช้เริ่มต้นระบบจะอยู่ที่ด้านบนของรายการ ในตัวอย่างนี้ Apple SSD เป็นดิสก์เริ่มต้นระบบ
  3. เลือกดิสก์ที่คุณต้องการลบ ไม่เห็นดิสก์ของคุณใช่ไหม 
  4. คลิกลบข้อมูล แล้วป้อนข้อมูลรายการเหล่านี้
    • ชื่อ: พิมพ์ชื่อที่คุณต้องการตั้งให้ดิสก์หลังจากที่คุณลบ
    • รูปแบบ: เลือก APFS หรือ Mac OS แบบขยาย (บันทึก) ยูทิลิตี้ดิสก์จะแสดงรูปแบบที่สามารถใช้งานร่วมกันได้ตามค่าเริ่มต้น
    • แบบแผน: เลือกผังพาร์ติชั่น GUID
  5. คลิกลบข้อมูลเพื่อเริ่มลบดิสก์ของคุณ รวมถึงตัวบรรจุและดิสก์โวลุ่มทั้งหมดภายในนั้น ระบบอาจขอให้คุณป้อน Apple ID ของคุณ ลืม Apple ID ของคุณหรือไม่ 
  6. เมื่อเสร็จแล้ว ให้ออกจากยูทิลิตี้ดิสก์
  7. หากคุณต้องการให้ Mac ของคุณสามารถเริ่มต้นระบบจากดิสก์ที่คุณลบได้ ให้ติดตั้ง macOS อีกครั้งบนดิสก์

วิธีลบดิสก์โวลุ่มบนดิสก์ของคุณ

  1. เริ่มต้นระบบจากการกู้คืน macOS จากนั้นเลือกยูทิลิตี้ดิสก์จากหน้าต่างยูทิลิตี้ แล้วคลิกดำเนินการต่อ
    หากคุณไม่ได้กำลังลบดิสก์โวลุ่มที่ Mac ของคุณใช้เริ่มต้นระบบ คุณไม่จำเป็นต้องเริ่มต้นระบบจากการกู้คืน macOS แต่เพียงแค่เปิดยูทิลิตี้ดิสก์จากโฟลเดอร์ยูทิลิตี้ในโฟลเดอร์แอพพลิเคชั่นของคุณ 
  2. ในแถบด้านข้างของยูทิลิตี้ดิสก์ ให้เลือกดิสก์โวลุ่มที่คุณต้องการลบ ดิสก์โวลุ่มที่ Mac ของคุณใช้เริ่มต้นระบบมีชื่อว่า Macintosh HD เว้นแต่คุณจะเปลี่ยนเป็นชื่ออื่น ไม่เห็นดิสก์โวลุ่มของคุณใช่ไหม
  3. คลิกลบข้อมูล แล้วป้อนข้อมูลรายการเหล่านี้
    • ชื่อ: พิมพ์ชื่อที่คุณต้องการตั้งให้ดิสก์โวลุ่มหลังจากที่คุณลบ
    • รูปแบบ: เลือก APFS หรือ Mac OS แบบขยาย (บันทึก) ยูทิลิตี้ดิสก์จะแสดงรูปแบบที่สามารถใช้งานร่วมกันได้ตามค่าเริ่มต้น
  4. หากคุณเห็นปุ่มลบกลุ่มดิสก์โวลุ่ม แสดงว่าดิสก์โวลุ่มที่คุณเลือกเป็นส่วนหนึ่งของกลุ่มดิสก์โวลุ่ม ในกรณีนี้ คุณควรลบกลุ่มดิสก์โวลุ่ม มิฉะนั้น ให้คลิกลบข้อมูลเพื่อลบเฉพาะดิสก์โวลุ่มที่เลือก ระบบอาจขอให้คุณป้อน Apple ID ของคุณ ลืม Apple ID ของคุณหรือไม่
  5. เมื่อเสร็จแล้ว ให้ออกจากยูทิลิตี้ดิสก์
  6. หากคุณต้องการให้ Mac ของคุณสามารถเริ่มต้นระบบจากดิสก์โวลุ่มที่คุณลบได้ ให้ติดตั้ง macOS อีกครั้งบนดิสก์โวลุ่มนั้น

เหตุผลที่ควรลบ

คุณสามารถลบได้ตลอดเวลา รวมถึงในกรณีต่อไปนี้ด้วย

  • คุณต้องการลบคอนเทนต์ทั้งหมดออกจาก Mac ของคุณอย่างถาวรและคืนค่าเป็นการตั้งค่าจากโรงงาน นี่เป็นหนึ่งในขั้นตอนสุดท้ายก่อนที่จะขาย ยกให้ผู้อื่น หรือนำ Mac ของคุณมาแลก
  • คุณจะเปลี่ยนรูปแบบของดิสก์ เช่น จาก รูปแบบ PC (FAT, ExFAT หรือ NTFS) เป็นรูปแบบ Mac (APFS หรือ Mac OS แบบขยาย)
  • คุณได้รับข้อความแจ้งว่าดิสก์ของคุณไม่สามารถอ่านได้โดยคอมพิวเตอร์เครื่องนี้
  • คุณกำลังพยายามแก้ไขปัญหาดิสก์ที่ยูทิลิตี้ดิสก์ไม่สามารถซ่อมได้
  • ตัวติดตั้ง macOS ไม่เห็นดิสก์ของคุณหรือไม่สามารถติดตั้งบนดิสก์ได้ ตัวอย่างเช่น ตัวติดตั้งอาจแจ้งว่าดิสก์ของคุณไม่ได้ฟอร์แมตอย่างถูกต้อง, ไม่ได้ใช้แบบแผนพาร์ติชั่น GUID, มีระบบปฏิบัติการเวอร์ชั่นที่ใหม่กว่า หรือไม่สามารถใช้ดิสก์เพื่อเริ่มต้นระบบคอมพิวเตอร์ของคุณได้
  • ตัวติดตั้ง macOS แจ้งว่าคุณอาจไม่ได้ติดตั้งไปที่โวลุ่มนี้เนื่องจากเป็นส่วนหนึ่งของ Apple RAID

เกี่ยวกับ APFS และ Mac OS แบบขยาย

ยูทิลิตี้ดิสก์ใน macOS High Sierra หรือใหม่กว่าสามารถลบได้โดยใช้รูปแบบ APFS (Apple File System) ที่ใหม่กว่าหรือรูปแบบ Mac OS แบบขยายรุ่นก่อนหน้า และจะเลือกรูปแบบที่เข้ากันได้สำหรับคุณโดยอัตโนมัติ

วิธีเลือกระหว่าง APFS และ Mac OS แบบขยาย

ยูทิลิตี้ดิสก์จะพยายามตรวจจับประเภทของพื้นที่จัดเก็บข้อมูล แล้วแสดงรูปแบบที่เหมาะสมในเมนูรูปแบบ หากตรวจจับประเภทไม่ได้ ให้เลือก Mac OS แบบขยายซึ่งทำงานได้กับ macOS ทุกเวอร์ชั่น หากคุณต้องการเปลี่ยนรูปแบบ ให้ตอบคำถามต่อไปนี้

  • คุณกำลังจะฟอร์แมตดิสก์ที่ติดตั้งใน Mac ใช่หรือไม่
    หากดิสก์ในตัวมาพร้อมกับ APFS ที่จัดรูปแบบแล้ว ยูทิลิตี้ดิสก์จะแนะนำ APFS อย่าเปลี่ยนเป็น Mac OS แบบขยาย
  • คุณกำลังจะติดตั้ง macOS High Sierra หรือใหม่กว่าเป็นครั้งแรกบนดิสก์ใช่หรือไม่
    หากคุณต้องการลบดิสก์ก่อนที่จะติดตั้ง High Sierra หรือใหม่กว่าเป็นครั้งแรกบนดิสก์นั้น ให้เลือก Mac OS แบบขยาย (บันทึก) ระหว่างการติดตั้ง ตัวติดตั้ง macOS จะตัดสินใจว่าจะแปลงดิสก์เป็นรูปแบบ APFS โดยอัตโนมัติโดยไม่ต้องลบไฟล์ของคุณหรือไม่
  • คุณกำลังจัดเตรียมดิสก์สำรองข้อมูล Time Machine หรือตัวติดตั้งที่สามารถบูตได้ใช่หรือไม่
    เลือก Mac OS แบบขยาย (บันทึก) สำหรับดิสก์ที่คุณวางแผนว่าจะใช้เป็นดิสก์ข้อมูลสำรอง Time Machine หรือเป็นตัวติดตั้งที่สามารถบูตได้
  • คุณจะใช้ดิสก์กับ Mac เครื่องอื่นหรือไม่
    หาก Mac เครื่องอื่นไม่ได้ใช้ macOS High Sierra หรือใหม่กว่า ให้ใช้ Mac OS แบบขยาย (บันทึก) macOS เวอร์ชั่นก่อนหน้าไม่สามารถใช้งานกับดิสก์โวลุ่มที่ถูกฟอร์แมตแบบ APFS ได้

วิธีระบุรูปแบบที่ใช้อยู่ในปัจจุบัน

หากต้องการดูว่ากำลังใช้รูปแบบใดอยู่ในปัจจุบัน ให้ใช้หนึ่งในวิธีการต่อไปนี้

  • เลือกดิสก์โวลุ่มในแถบด้านข้างของยูทิลิตี้ดิสก์ แล้วตรวจสอบข้อมูลที่แสดงทางขวา สำหรับรายละเอียดเพิ่มเติม ให้เลือกไฟล์ > ขอรายละเอียดจากแถบเมนูยูทิลิตี้ดิสก์
  • เปิดข้อมูลระบบแล้วเลือกพื้นที่จัดเก็บข้อมูลในแถบด้านข้าง คอลัมน์ระบบไฟล์ทางขวาแสดงรูปแบบของแต่ละโวลุ่ม
  • เลือกโวลุ่มใน Finder แล้วเลือกไฟล์ > ขอรายละเอียดจากแถบเมนู หน้าต่างขอรายละเอียดแสดงรูปแบบของโวลุ่มนั้น

หากดิสก์หรือดิสก์โวลุ่มของคุณไม่ปรากฏขึ้นหรือลบไม่สำเร็จ

  1. ปิดเครื่อง Mac ของคุณ แล้วถอดอุปกรณ์ที่ไม่จำเป็นทั้งหมดออกจาก Mac ของคุณ
  2. หากคุณกำลังลบไดรฟ์ภายนอก ตรวจสอบให้แน่ใจว่าไดรฟ์นั้นเชื่อมต่อโดยตรงกับ Mac ของคุณโดยใช้สายที่มีประวัติทำงานปกติ จากนั้นให้ปิดไดรฟ์ แล้วเปิดใหม่อีกครั้ง
  3. หากดิสก์หรือดิสก์โวลุ่มของคุณยังคงไม่ปรากฏในยูทิลิตี้ดิสก์ หรือยูทิลิตี้ดิสก์รายงานว่ากระบวนการลบล้มเหลว ดิสก์หรือ Mac ของคุณอาจต้องเข้ารับบริการซ่อม หากคุณต้องการความช่วยเหลือ โปรดติดต่อฝ่ายบริการช่วยเหลือของ Apple

ดูข้อมูลเพิ่มเติม

  • หากคุณไม่สามารถเริ่มต้นระบบจากการกู้คืน macOS ได้ คุณสามารถใช้ดิสก์เริ่มต้นระบบอื่นแทนได้
  • หากยูทิลิตี้ดิสก์แสดงปุ่มตัวเลือกการรักษาความปลอดภัยในหน้าต่างลบข้อมูล คุณสามารถคลิกปุ่มนั้นเพื่อเลือกระหว่างการลบที่เร็วกว่า (แต่ปลอดภัยน้อยกว่า) และการลบที่ช้ากว่า (แต่ปลอดภัยมากกว่า) ได้ ยูทิลิตี้ดิสก์เวอร์ชั่นเก่าบางเวอร์ชั่นมีตัวเลือกในการทำข้อมูลทั้งหมดให้เป็นศูนย์แทน ตัวเลือกการลบที่ปลอดภัยเหล่านี้ไม่มีให้บริการหรือจำเป็นสำหรับโซลิดสเตทไดรฟ์ (SSD) และพื้นที่เก็บข้อมูลแบบแฟลช
  • วิธีลบ iPhone, iPad หรือ iPod touch ของคุณ
