ความพร้อมใช้งานของการตรวจสอบสิทธิ์สองปัจจัยสำหรับ Apple ID

การตรวจสอบสิทธิ์สองปัจจัยเป็นการรักษาความปลอดภัยที่เพิ่มขึ้นอีกชั้นหนึ่งสำหรับ Apple ID ซึ่งออกแบบมาเพื่อให้มั่นใจได้ว่าคุณเป็นเพียงผู้เดียวที่สามารถเข้าถึงบัญชีของคุณได้ แม้ว่าจะมีบุคคลอื่นที่รู้รหัสผ่านของคุณก็ตาม 

วิธีเปิดการตรวจสอบสิทธิ์สองปัจจัยสำหรับ Apple ID ของคุณ

คุณสามารถใช้การตรวจสอบสิทธิ์สองปัจจัยกับ Apple ID ของคุณ หากคุณมีอุปกรณ์ที่ใช้ iOS หรือ macOS เวอร์ชั่นล่าสุด หรือหากคุณเข้าใช้เว็บเบราเซอร์และหมายเลขโทรศัพท์ได้* ดูวิธีเปิดการตรวจสอบสิทธิ์สองปัจจัยบน iPhone, iPad, iPod touch หรือ Mac ของคุณ และดูข้อมูลความต้องการของระบบที่แนะนำ

แม้ว่าผู้ใช้ iCloud ส่วนใหญ่จะสามารถเปิดการตรวจสอบสิทธิ์สองปัจจัยได้แล้ว แต่บัญชีบางประเภทอาจยังคงไม่เข้าเกณฑ์ในการใช้งานตามดุลยพินิจของ Apple

* เรียนรู้เพิ่มเติมเกี่ยวกับเหตุผลที่คุณต้องมีหมายเลขโทรศัพท์ที่เชื่อถือได้สำหรับการตรวจสอบสิทธิ์สองปัจจัย

การตรวจสอบสิทธิ์สองปัจจัยสำหรับ Apple ID สามารถใช้งานได้ที่ใดบ้าง

คุณสามารถตั้งค่าการตรวจสอบสิทธิ์สองปัจจัยได้ในประเทศหรือภูมิภาคเหล่านี้ เมื่อเพิ่มประเทศหรือภูมิภาคเพิ่มเติม ลิงก์สำหรับเปิดการตรวจสอบสิทธิ์สองปัจจัยจะปรากฏขึ้นมาโดยอัตโนมัติในอุปกรณ์ที่ใช้งานร่วมกันได้
