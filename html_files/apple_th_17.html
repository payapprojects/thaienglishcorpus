กู้คืน Mac ของคุณจากข้อมูลสำรอง

เรียนรู้วิธีถ่ายโอนไฟล์ทั้งหมดจากข้อมูลสำรอง Time Machine ของคุณไปยัง Mac

หากคุณเคยใช้ Time Machine เพื่อสร้างข้อมูลสำรองสำหรับ Mac คุณสามารถกู้คืนไฟล์จากข้อมูลสำรองหรือจากสแนปช็อตภายในเครื่องบนดิสก์เริ่มต้นระบบของคุณได้ คุณอาจต้องการกู้คืนไฟล์หลังจากไฟล์ต้นฉบับถูกลบออกจาก Mac หรือฮาร์ดดิสก์ (หรือ SSD) ใน Mac ของคุณถูกลบหรือเปลี่ยน เช่น ระหว่างการซ่อม

กู้คืนจากข้อมูลสำรองของ Time Machine

เมื่อคุณกู้คืนจากข้อมูลสำรอง Time Machine คุณสามารถเลือกกู้คืนไฟล์ทั้งหมดของคุณหรือกู้คืนทั้งระบบปฏิบัติการ Mac (macOS) และไฟล์ทั้งหมดของคุณ

กู้คืนไฟล์ของคุณทั้งหมด

  1. ตรวจสอบให้แน่ใจว่าดิสก์ข้อมูลสำรอง Time Machine ของคุณเชื่อมต่อกับ Mac แล้วและเปิดอยู่ จากนั้นเปิด Mac ของคุณ
  2. เปิดผู้ช่วยการโยกย้ายซึ่งอยู่ในโฟลเดอร์ยูทิลิตี้ของโฟลเดอร์แอพพลิเคชั่น 
    • หาก Mac เริ่มต้นระบบไปยังผู้ช่วยตั้งค่าที่ขอรายละเอียดต่างๆ เช่น ประเทศ คีย์บอร์ด และเครือข่ายของคุณ ให้ดำเนินการต่อไปยังขั้นตอนถัดไป ผู้ช่วยการตั้งค่ารวมถึงผู้ช่วยการโยกย้าย
    • หาก Mac ไม่เริ่มต้นระบบเลย หรือคุณต้องการกู้คืน macOS ที่คุณใช้เมื่อสร้างข้อมูลสำรอง ให้ทำตามขั้นตอนเพื่อกู้คืนทั้ง macOS และไฟล์ของคุณ
  3. เมื่อระบบถามว่าคุณต้องการถ่ายโอนข้อมูลของคุณอย่างไร ให้เลือกตัวเลือกเพื่อถ่ายโอนจาก Mac, ข้อมูลสำรอง Time Machine หรือดิสก์เริ่มต้นระบบ จากนั้นคลิกดำเนินการต่อ
  4. เลือกข้อมูลสำรองของ Time Machine แล้วคลิกดำเนินการต่อ
  5. หากระบบขอให้คุณเลือกจากรายการข้อมูลสำรองที่เรียงตามวันที่และเวลา ให้เลือกข้อมูลสำรองแล้วคลิกดำเนินการต่อ
  6. เลือกข้อมูลที่ต้องการถ่ายโอน แล้วคลิกดำเนินการต่อเพื่อเริ่มถ่ายโอน หน้าจอนี้อาจดูแตกต่างไปบน Mac ของคุณ ดังต่อไปนี้
  7. หากคุณมีเนื้อหาจำนวนมาก การถ่ายโอนอาจใช้เวลาหลายชั่วโมงกว่าจะเสร็จสิ้น เมื่อการถ่ายโอนเสร็จสมบูรณ์ ให้รีสตาร์ท Mac ของคุณแล้วเข้าสู่ระบบในบัญชีที่โยกย้ายเพื่อดูไฟล์
      

กู้คืนทั้ง macOS และไฟล์ของคุณ

ขั้นตอนเหล่านี้จะลบฮาร์ดดิสก์ของคุณแล้วใช้ข้อมูลสำรองเพื่อกู้คืนทั้งไฟล์และ macOS เวอร์ชั่นเฉพาะที่คุณใช้สร้างข้อมูลสำรอง

  1. ตรวจสอบให้แน่ใจว่าดิสก์ข้อมูลสำรอง Time Machine ของคุณเชื่อมต่อแล้วและเปิดอยู่
    หากดิสก์ข้อมูลสำรองของคุณไม่พร้อมใช้งาน ให้ทำต่อไป: คุณอาจสามารถกู้คืนจากสแนปช็อตภายในเครื่องบนดิสก์เริ่มต้นระบบของคุณ
  2. เปิด Mac ของคุณ แล้วกด Command (⌘)-R ค้างไว้ทันทีเพื่อเริ่มต้นระบบจากการกู้คืน macOS ปล่อยปุ่มเมื่อคุณเห็นโลโก้ Apple หรือลูกโลกหมุน
  3. เมื่อคุณเห็นหน้าต่างยูทิลิตี้ macOS ให้เลือกตัวเลือกเพื่อกู้คืนจากข้อมูลสำรอง Time Machine
  4. คลิกดำเนินการต่อจนกว่าระบบจะขอให้คุณเลือกแหล่งข้อมูลกู้คืน จากนั้นเลือกดิสก์ข้อมูลสำรอง Time Machine ของคุณ หรือเลือกดิสก์เริ่มต้นระบบ (Macintosh HD) ซึ่งอาจมีสแนปช็อตภายในเครื่องที่คุณสามารถใช้กู้คืนได้
  5. คลิกดำเนินการต่อ หากดิสก์ของคุณถูกเข้ารหัส ระบบจะขอให้คุณปลดล็อคดิสก์: ป้อนรหัสผ่านผู้ดูแลระบบที่คุณใช้เมื่อตั้งค่า Time Machine จากนั้นคลิกดำเนินการต่ออีกครั้ง
  6. เลือกข้อมูลสำรอง หากมี แล้วคลิกดำเนินการต่อ 
  7. เลือกดิสก์ปลายทาง ซึ่งจะรับเนื้อหาข้อมูลสำรองของคุณ หากกู้คืนจากสแน็ปช็อตภายในเครื่อง ระบบจะไม่ขอให้คุณเลือกปลายทาง 
  8. คลิกกู้คืนหรือดำเนินการต่อ หาก Mac ของคุณเปิด FileVault อยู่ ระบบจะขอให้คุณป้อนรหัสผ่านผู้ดูแลระบบเพื่อปลดล็อคดิสก์
  9. เมื่อเสร็จสิ้นแล้ว ให้รีสตาร์ท Mac
      

กู้คืนไฟล์บางไฟล์

ดูวิธีการใช้ Time Machine เพื่อกู้คืนไฟล์บางไฟล์ ซึ่งรวมถึงไฟล์เวอร์ชั่นเก่าของคุณ


ดูข้อมูลเพิ่มเติม

  • วิธีอื่นที่ใช้ในการสำรองข้อมูลและกู้คืนไฟล์
  • สิ่งที่ควรทำหากคุณไม่สามารถกู้คืนด้วย Time Machine หากคุณต้องการความช่วยเหลือ ให้ติดต่อฝ่ายบริการช่วยเหลือของ Apple
